# c-university-project-store-database-
Basically a project I did for my final grade in basic coding with c++. Finished in September 2025, might come back to it one day.

What does the app do? </br>
It's a basic console app, used to store and manage the inventory of a fictional game store (adding/deleting/editing items, filtering; has a functional trash bin with options to restore items or delete them permamently). Written in Polish, with honestly no intention to translate it.

How to use it? </br>
The .exe is in the 'output' folder. </br>
Every option is available on screen, just push the corresponding button (displayed on the left)</br>
There is also a ready database if you don't want to waste time adding your own items - just go through opreacje na głównej bazie > odczyt z pliku
